# Affari Tuoi 🎁

Un mini-gioco web ispirato al programma TV "Affari Tuoi" (il gioco dei pacchi).
È un **unico file** (`index.html`): musica e tutto il resto sono già incorporati.

## Come si gioca
1. Scegli il tuo pacco tra i 20 (uno per ogni regione d'Italia).
2. Apri gli altri pacchi: ogni tanto arriva la telefonata del **Dottore** con un'**offerta in denaro** o lo **scambio**.
3. Quando apri un **pacco blu** (valore basso) è una buona notizia, e parte una **canzone a caso**. 🎵
   Tra i pacchi blu ci sono anche i premi scherzo: 🐕 Genarino, 💃 la Ballerina e 🪨 la Pietra.
4. Con gli ultimi 2 pacchi scegli: **Apri il tuo pacco**, **Cambia pacco** o **La Regione Fortunata** 🇮🇹.

## La Regione Fortunata
- 1° tentativo (tra 20) → **100.000 €**
- 2° tentativo (tra 10) → **50.000 €**
- Sbagli entrambi → niente premio

## Pubblicare su GitHub Pages
1. Carica **solo `index.html`** nel repository (la musica è già dentro).
2. **Settings → Pages**, branch `main`, cartella `/root`.
3. Online su `https://<utente>.github.io/<repo>/`.
